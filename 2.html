    • git branch -d or --delete [branch-name]

    git branch -d or --delete [branch-name] 특정 이름을 가진 브랜치 삭제. 삭제하고 싶은 브랜치 위에 있는 경우에 삭제가 되지 않는다. 다른 브랜치로 이동 후 삭제해야 한다. 또한 브랜치 삭제는 완전히 병합된 브랜치만 삭제가 가능하다.
